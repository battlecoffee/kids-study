# 英文單字考前複習

小孩英文單字的考前複習頁，筆電投電視用。單一靜態 HTML，零建置、零後端。

## 用法

直接用瀏覽器開 `index.html`，按 `F11` 全螢幕投到電視即可。離線也能跑（字體與語音會 fallback 到系統內建）。

三種模式：

- **複習** —— 全部攤開帶著看。
- **測驗** —— 中文遮起來，點卡翻開並唸出單字＋拼字。
- **隨機抽考** —— 全螢幕一次跳一張，最接近真實考試。`Space` 看答案 / `Enter`·`→` 下一張 / `S` 再聽。

考試形式：老師指英文單字 → 唸發音 → 逐字母拼 → 說中文。

## 語音

用瀏覽器原生語音，會唸英文單字＋逐字母（中文不唸，留給小孩自己說）。右上 🔊 選單可換語音，預設優先 Samantha。想要更自然的發音，在 macOS「系統設定 → 輔助使用 → 朗讀內容 → 管理語音」下載英文的 Enhanced／Premium 版。

## 開發

```bash
npm install
npm test        # jsdom 回歸測試
```

加單字、加類別、設計約束等維護細節見 `CLAUDE.md`。

## 部署

GitHub Pages：`index.html` 放根目錄，Settings → Pages → main/root。
