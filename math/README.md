# 數學練習 🔢

給低年級小朋友的數學練習互動網頁（注音 + 音效 + 答題語音鼓勵）。

**線上玩**：https://battlecoffee.github.io/kids-study/math/

## 結構（單一資料夾，每個主題各自是一份離線單頁）

```
math/
├── index.html        ← 主題選單（投電視進來先選主題）
├── 錢幣闖關.html      ← 錢幣練習
└── 日期小高手.html    ← 日期 / 月曆練習
```

進入順序：根首頁 → 數學 → 選主題。各主題頁左上角可「⬅️ 數學」回選單。

## 🪙 錢幣闖關（每關隨機 5 題，可無限重玩）
- 👀 認幣值　🔄 換錢（同值換算）　🧮 數錢　🛒 陀螺商店（錢夠不夠）

## 📅 日期小高手（月曆互動，每題換月份）
- 前一天　隔天　找星期（可切換只練某一種）

## 共同特色
- 全畫面注音，低年級看得懂
- **題目不朗讀**，由孩子自行閱讀（增進閱讀能力）；只在答對 / 鼓勵時出聲
- 答錯不扣分、可重試、有提示
- 每份都是單一 HTML、離線可用
